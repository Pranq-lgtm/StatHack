## 🚀 StatHack: Machine Learning & Deep Learning Comparison

This repository contains the source code and model evaluations developed for the **StatHack** competition. The project focuses on improving predictive performance by transitioning from traditional machine learning algorithms to deep learning architectures.

### 📊 Performance Summary

We implemented and compared two distinct approaches to solve the problem set:

* **Baseline Model (KNN):** Implemented in `StatHack.ipynb`, utilizing K-Nearest Neighbors. This served as our performance benchmark.
* **F1 Score:** 0.03463


* **Optimized Model (CNN):** Implemented in `StatHack_2.ipynb`, utilizing a Convolutional Neural Network (CNN) for better feature extraction and classification.
* **F1 Score:** 0.1193
* **Final Accuracy:** 46.34%



### 📈 Key Results

By migrating from a KNN architecture to a CNN-based approach, we achieved a **244.50% increase** in our evaluation score. This highlights the effectiveness of deep learning in handling the complexity of the StatHack dataset.
